    Model

    The engineering model sums each branch capacitance, calculates S1-S5, then uses bal = sqrt((S1 + S2 + 2S3)^2 + 3(S4 - S5)^2). The double-star neutral unbalance current is 0.001 * V * 2πf * bal / (2X), shown in mA. Auto mode stops at the smallest swap count whose displayed current rounds to 0.00 mA.
