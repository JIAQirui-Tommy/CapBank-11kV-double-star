    Calculation Details

    Model

    The engineering model sums each branch capacitance, calculates S1-S5, then uses bal = sqrt((S1 + S2 + 2S3)^2 + 3(S4 - S5)^2). The double-star neutral unbalance current is 0.001 * V * 2πf * bal / (2X), shown in primary-side mA. Secondary-side current is primary current / CT ratio. Auto mode checks up to 4 swap pairs, skips same-value swaps, and stops at the smallest swap count whose displayed primary current rounds to 0.00 mA.
